Документация Python Wikipedia

Функции и Классы

wikipedia.search(query, results = 10, suggestion = False)

Выполняет поиск в Википедии по запросу (query)

Параметры:
results : int

Максимальное количество возвращенных результатов.

suggestion : bool

Если значение True, вернёт результат и предложения (если таковые имеются) в виде кортежа.

wikipedia.suggest(query)

Возвращает предложение для поиска в Википедии по запросу query. Возвращает строку или None, если предложение не найдено.

wikipedia.summary(query, sentences=0, chars=0, auto_suggest=True, redirect=True)

Краткое описание страницы в виде обычного текста.

Параметры:
sentences : int

если задано, вернёт первые предложения (их может быть не более 10).

chars : int

если задано, возвращайте только первые символы символов (фактический возвращаемый текст может быть немного длиннее).

auto_suggest : bool

пусть Википедия найдет допустимое название страницы для запроса

redirect : bool

разрешить перенаправление, не вызывая RedirectError.

wikipedia.page(title=None, pageid=None, auto_suggest=True, redirect=True, preload=False)

Получите объект WikipediaPage для страницы с заголовком title или pageid pageid (взаимоисключающие).

Параметры:
title : Any

заголовок страницы для загрузки

pageid : Any

числовой идентификатор страницы для загрузки

auto_suggest : bool

пусть Википедия найдет допустимое название страницы для запроса

redirect : bool

разрешить перенаправление, не вызывая RedirectError

preload : bool

загружайте содержимое, сводку, изображения, ссылки и ссылки во время инициализации

wikipedia.geosearch(latitude, longitude, title=None, results=10, radius=1000)

Выполните геопоиск в Википедии по широте и долготе, используя HTTP API, описанный в http://www.mediawiki.org/wiki/Extension:GeoData

Параметры:
latitude : float or decimal.Decimal

Широта

longitude : float or decimal.Decimal

Долгота

title : Any

Название статьи для поиска

results : int

максимальное количество возвращаемых результатов

radius : int

Радиус поиска в метрах. Значение должно быть в диапазоне от 10 до 10000

class wikipedia.WikipediaPage(title=None, pageid=None, redirect=True, preload=False, original_title=u'')

Содержит данные со страницы Википедии. Использует методы свойств для фильтрации данных из необработанного HTML-кода.

Параметры:
categories

Список категорий страницы.

content

Текстовое содержимое страницы, за исключением изображений, таблиц и других данных.

coordinates

Кортеж десятичных знаков в виде (lat, lon) или None

html()

Получить полный HTML-код страницы.

Это может быть довольно медленным на длинных страницах.
images

Список URL-адресов изображений на странице.

links

Список названий ссылок на страницы Википедии на странице.

Включает только статьи из пространства имен 0, что означает отсутствие категории, обсуждения пользователей или других мета-страниц Википедии.
parent_id

Идентификатор редакции родительской версии текущей редакции этой страницы. Дополнительные сведения см. в разделе revision_id.

references

Список URL-адресов внешних ссылок на странице. Может содержать внешние ссылки внутри страницы, которые технически нигде не цитируются.

revision_id

Идентификатор редакции страницы.

Идентификатор редакции - это число, которое однозначно идентифицирует текущую версию страницы. Его можно использовать для создания постоянной ссылки или для других прямых вызовов API. Дополнительные сведения см. в разделе Справка: История страниц.

section(section_title)

Получите текстовое содержимое раздела из self.sections. Возвращает None, если section_title не найден, в противном случае возвращает строку без пробелов.

Это удобный метод, который оборачивает self.content.

Вызов section для раздела, имеющего подзаголовки, не вернет полный текст всех подразделов. Он получает только текст между section_title и следующим подзаголовком, который часто бывает пустым.
sections

Список названий разделов из оглавления на странице.

summary

Краткое описание страницы в виде обычного текста.

revision_id

Получить полный HTML-код страницы.

wikipedia.languages()

все поддерживаемые в настоящее время языковые префиксы (обычно код языка ISO).

Может быть введен в set_lang для изменения Mediawiki, из которого википедия запрашивает результаты.

Возвращает: dict пар prefix: local_lang_name. Чтобы получить просто список префиксов, воспользуйтесь wikipedia.languages().keys().

wikipedia.set_lang(prefix)

Измените язык запрашиваемого API. Установите префикс на один из двух буквенных префиксов, найденных в списке всех Википедий.

После настройки языка кэш для поиска, предложений и сводки будет очищен.

wikipedia.set_rate_limiting(rate_limit, min_wait=datetime.timedelta(0, 0, 50000))

Включить или отключить ограничение скорости запросов к серверам Mediawiki. Если ограничение скорости не включено, при некоторых обстоятельствах (в зависимости от загрузки Википедии, количества запросов, которые делаете вы и другие пользователи википедии, и других факторов) Википедия может возвращать ошибку HTTP timeout.

Включение ограничения скорости обычно предотвращает эту проблему, но, пожалуйста, обратите внимание, что HTTPTimeoutError все еще может быть вызван.

Параметры:
rate_limit : bool

следует ли включать ограничение скорости или нет

min_wait : timedelta

Если включено ограничение скорости, min_wait - это timedelta, описывающая минимальное время ожидания перед запросами. По умолчанию используется значение timedelta(milliseconds=50).

wikipedia.random(pages=1)

Получите список случайных названий статей в Википедии.

Параметры:
pages : int

количество возвращенных случайных страниц (не более 10)

wikipedia.donate()

Откроет страницу пожертвований Wikimedia в вашем любимом браузере.

Глобальные классы исключений и предупреждений википедии.

exception wikipedia.exceptions.DisambiguationError(title, may_refer_to)

Исключение возникает, когда страница преобразуется в страницу устранения неоднозначности.

Свойство options содержит список названий страниц Википедии, на которые может ссылаться запрос.

exception wikipedia.exceptions.HTTPTimeoutError(query)

Исключение возникает, когда время ожидания запроса к серверам Mediawiki истекает.

exception wikipedia.exceptions.PageError(pageid=None, *args)

Исключение возникает, когда ни одна Википедия не соответствует запросу.

exception wikipedia.exceptions.RedirectError(title)

Исключение возникает, когда заголовок страницы неожиданно преобразуется в перенаправление.

exception wikipedia.exceptions.WikipediaException(error)

Базовый класс исключений Википедии.
